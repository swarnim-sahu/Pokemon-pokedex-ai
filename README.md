# Pokémon Pokédex AI 
An AI-powered Pokémon image classifier built using Convolutional Neural Networks (CNN) and Streamlit.

# Features
* Upload a Pokémon image
* Predict Pokémon using trained CNN model
* Display Pokémon stats (HP, Attack, Defense, Speed, Type)
* Interactive UI using Streamlit

# Tech Stack
* Python
* TensorFlow / Keras
* NumPy
* Streamlit

# Project Structure
pokemon-pokedex-ai
-app.py
-train.py
-predict.py
-pokemon_model.h5
-requirements.txt

# Run Locally
pip install -r requirements.txt
streamlit run app.py

# Model Details
* CNN-based image classifier
* Input size: 64x64 images
* Classes: Bulbasaur, Charmander, Pikachu, Squirtle
